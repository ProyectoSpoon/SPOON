Crear Usuario de Prueba para Spoon Restaurant

Instrucciones

Nota: Este método es para desarrollo y pruebas. No debe usarse en producción.
Paso 1: Abre la página de login de Spoon Restaurant en http://localhost:3000/login
Paso 2: Abre la consola del navegador:
  • Chrome/Edge: Presiona F12 o Ctrl+Shift+J (Windows/Linux) o Cmd+Option+J (Mac)
  • Firefox: Presiona F12 o Ctrl+Shift+K (Windows/Linux) o Cmd+Option+K (Mac)
  • Safari: Activa el menú Desarrollador y selecciona "Mostrar Consola JavaScript"
Paso 3: Copia y pega el siguiente código en la consola: 
fetch('/test-user-script.js')
  .then(response => response.text())
  .then(script => {
    eval(script);
  })
  .catch(error => console.error('Error cargando el script:', error));
Paso 4: Ejecuta la función para crear el usuario de prueba escribiendo en la consola: 
createTestUser()
Paso 5: Espera a que el proceso termine. Verás un mensaje de confirmación en la consola.
Paso 6: Usa las siguientes credenciales para iniciar sesión:

Email: test@spoonrestaurant.com

Contraseña: Test2025!

Solución de problemas

Si recibes un error "auth/email-already-in-use": El usuario ya existe. Puedes usar las credenciales proporcionadas para iniciar sesión.

Si recibes otros errores: Verifica que la aplicación esté correctamente configurada con Firebase y que estés en la página de login.
